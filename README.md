###关于使用polymer

起步项目我放回static/components文件夹了

之前的那些文件还能直接用

*使用自己想要的Element*

1. 在[Elements集合](http://docs.polymerchina.org/docs/elements/)选择想要的Elemet
2. 点进选中的介绍页面，会出现一个GET XXX按钮，点开之后切换到Github标签
3. 复制绿色框里的命令，在static文件夹 右键>>Git Bash 点击窗口边框>>编辑>>粘贴 回车运行
4. 完~成~，在页面引用即可直接使用
