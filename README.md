##安装软件包

* nodejs
* python2.7
* Sublime Text 2
* msysgit
* mongodb
* nodejs组件见下面
###npm
安装完NodeJS之后应该可以在cmd里使用npm(Node Package Manager)安装需要的软件包

在cmd里输入
```
npm install express socket.io url ejs
```
就可以了
